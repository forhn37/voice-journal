# Voice Journal PRD (Product Requirements Document)

## 1. 프로덕트 비전

### 1.1 한줄 요약
> "말로 기록하면, AI가 그림일기로 만들어주는 감성 저널링 앱"

### 1.2 왜 만드는가?
- 일기 쓰고 싶은데 **귀찮음** → 말로 하면 쉬움
- 텍스트만 있으면 **재미없음** → 그림이 있으면 기억에 남음
- 혼자 사는 사람들에게 **위로**가 되는 앱

### 1.3 타겟 유저
- 혼자 사는 20-30대
- 일기 쓰고 싶지만 귀찮은 사람
- 감정 기록에 관심 있는 사람
- MBTI I 성향 (내향형)

## 2. 핵심 기능

### 2.1 음성 일기 작성

#### 플로우
```
[메인 화면] → [녹음 버튼] → [녹음 중] → [녹음 완료]
     → [로딩 화면 - 캐릭터] → [결과 화면 - 그림일기]
```

### 2.2 AI 처리 + 로딩 경험

#### 로딩 시퀀스 (15~30초)
| 시간 | 캐릭터 대사 | 상태 |
|-----|-----------|------|
| 0초 | "오늘 이야기 잘 들었어!" | STT 시작 |
| 3초 | "음... 그랬구나~" | STT 완료 |
| 6초 | "그림 그려볼게! 잠깐만~" | GPT 분석 |
| 9초 | "[감정 기반 공감 메시지]" | DALL-E 시작 |
| 12초 | "오 이거 잘 나온다~" | 생성 중 |
| 15초+ | "짜잔~! 🎉" | 완료 |

#### 감정별 공감 메시지
| 감정 | 메시지 예시 |
|-----|-----------|
| 기쁨 | "와~ 진짜?! 대박이다!! 🎉" |
| 슬픔 | "그랬구나... 힘들었겠다 😢" |
| 화남 | "헐 그건 진짜 화날 만해..." |
| 불안 | "걱정되는 거 당연해. 괜찮아" |
| 평범 | "오늘도 수고했어~ ☺️" |
