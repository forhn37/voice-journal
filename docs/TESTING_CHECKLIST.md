## 1. 전체 플로우 테스트

### 회원가입 → 온보딩 → 일기 작성 플로우

- [ ] **회원가입**
  - [ ] 이메일 로그인 정상 동작
  - [ ] 구글 로그인 정상 동작 (설정된 경우)
  - [ ] 잘못된 이메일 형식 에러 표시
  - [ ] 이미 가입된 이메일 에러 표시

- [ ] **온보딩 (신규 가입자)**
  - [ ] Step 1: 환영 화면 표시
  - [ ] Step 2: 닉네임 입력 (1-10자 제한)
  - [ ] Step 3: 알림 시간 선택
  - [ ] 온보딩 완료 후 메인 페이지로 이동
  - [ ] 재로그인 시 온보딩 스킵 (DB에 프로필 존재)

- [ ] **일기 작성 (메인 페이지)**
  - [ ] 녹음 버튼 표시
  - [ ] 마이크 권한 요청
  - [ ] 녹음 시작/정지 동작
  - [ ] 녹음 시간 표시 (3초 ~ 5분)
  - [ ] 3초 미만 녹음 시 에러 메시지
  - [ ] 5분 초과 시 자동 정지

- [ ] **AI 분석 및 이미지 생성**
  - [ ] 로딩 캐릭터 표시
  - [ ] 로딩 메시지 순차 표시
  - [ ] STT 완료 (음성 → 텍스트)
  - [ ] AI 분석 완료 (감정, 장면, 요약)
  - [ ] 이미지 생성 완료
  - [ ] 결과 화면 표시 (이미지 + 텍스트)

- [ ] **일기 저장**
  - [ ] 저장 버튼 동작
  - [ ] DB에 저장 확인
  - [ ] 저장 후 메인 페이지로 이동

- [ ] **사용량 제한**
  - [ ] 일일 3회 제한 동작
  - [ ] 3회 초과 시 안내 메시지
  - [ ] 다음 날 0시 이후 리셋 확인
